# Financial Fraud Detection Pipeline

An end-to-end machine learning pipeline designed to detect suspicious financial transactions by leveraging advanced data preprocessing, feature engineering, classification algorithms and real-time prediction APIs. The system is built with a production-oriented modular architecture focusing on scalability, model optimization and monitoring.

## Project Overview

This project simulates a real-world fintech fraud detection workflow where transactional data is processed, cleaned and analyzed to identify potentially fraudulent activities. The pipeline integrates machine learning models with deployment-ready API services for real-time fraud prediction.

## Features

- Automated transaction data cleaning and preprocessing  
- Advanced feature engineering for fraud pattern extraction  
- Handling class imbalance using resampling techniques  
- Random Forest based fraud classification model  
- Hyperparameter tuning for model optimization  
- Performance evaluation using accuracy, precision and recall metrics  
- REST API deployment using FastAPI for real-time predictions  
- Modular architecture for scalability and maintainability  
- Model performance monitoring and logging pipeline
